# RMS - Resource Management System

A monorepo project with React frontend and Express backend.

## Project Structure

```
root/
├── backend/        # Node.js + Express API
├── frontend/       # Vite + React
├── docs/          # Documentation
└── README.md
```

## Getting Started

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Tech Stack

- **Frontend**: Vite, React (JSX)
- **Backend**: Node.js, Express
